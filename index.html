Research / Projects

Edge Intelligence based Collaborative Learning approach for IoT Edge

Research domains - AI / IoT / Edge Computing

A research on adapting collaborative learning in resource constrained IoT devices to train deep learning models efficiently with less computational power and high energy efficiency. Python, Flask and MongoDB were utilized to implement the server component along with PyTorch library. The client was implemented using Angular and SocketIO. A RPi 4 and a RPi zero were utilized as the IoT devices for testing and evaluation.

Machine Learning approach for Network Intrusion Detection and Prevention System

Research domains - Machine Learning

A system was implemented to detect network intrusions in real-time with the capability of denying the source of the intrusion and notify/alert end-users about the intrusion. Python along with TensorFlow was used to build the network classifier and Angular, NodeJS and Redis/MongoDB databases were utilized to develop the client and server components.

TrainBuzz - Crowdsourced IoT Tracking System for Trains

Research domains - IoT / Crowdsource

A crowdsourced IoT application to locate trains in real time. The concept was to gather data from passengers in the train and IoT trackers designed using Arduino and GSM/GPS shield that could be installed in the train to build up a real time navigation system. A mobile application was developed to demonstrate the system. The project got featured in the Cutting Edge exhibition in 2018.
